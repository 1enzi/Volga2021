Комплект артефактов по приложению:
1) Программный код приложения в формате .txt (Код приложения_Карпенко.txt);
2) zip-архив с данными приложения (Волга2021_Карпенко.zip);
3) Readme-файл (Readme.txt).

В комплекте артефактов по данному приложению будет представлен zip-архив, содержащий все данные приложения. Также в данном архиве будет представлен файл для открытия приложения через редактор Visual Studio. Это необходимо для того, чтобы обеспечить корректное создание логов приложением при проверке его работоспособности.

Инструкция по запуску приложения.

1) Запустить файл ConsoleApp1.exe (распологается в папке: C:\Users\Lenzi\source\repos\ConsoleApp1\ConsoleApp1\bin\Debug\netcoreapp3.1).
2) Будет открыто окно командной строки. В данном окне будет необходимо ввести путь к HTML-файлу. Программа проверят существование файла по введённому адресу, и если путь к файлу введён не корректно, то потребуется ввести его заново.
3) После ввода корректного пути расположения файла, пользователю будет предложено выбрать количество выводимых слов (к примеру, если в файле имеется 1000 различных слов, пользователь может вывести только 5 самых наиболее встречающихся из них). При вводе значения, превышаюшего количество слов, будут выведены все слова из массива.
4) После выполнения всех вышеописанных действий програма завершит работу.

Логирование в приложении. 

Логирование в приложении осуществляется с использованием библиотеки NLog. Файл с логами располагается в папке ...\ConsoleApp1\ConsoleApp1\bin\Debug\netcoreapp3.1\logs. Файл имеет имя вида "гггг-мм-дд" (короткий формат даты).
Логирование происходит при:
1) Запуске программы.
2) Вводе некорректного значения пользоваталем.
3) Завершении работы программы.
Для успешного логирования приложение должно иметь доступ к файлам Nlog.config и Nlog.xsd.
